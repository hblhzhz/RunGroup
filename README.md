# 跑团-组团健康跑步

跑团-组团健康跑步是一款专为跑步爱好者们开发的软件，
软件内提供丰富的跑团供您选择，跑团陪着你跑，您可以选择创建或者加入跑团，定期组队跑步。软件还可以记录每次跑步数据，让您看得到每次运动的数据变化。还提供了三种计时模式跑步，按照您的需求，陪您一同完成锻炼
